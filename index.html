Tentang Saya

Halo! saya Iwan Haryatno, orang-orang biasa memanggil saya Iwan. Saya sekarang tinggal di desa Windujaya yang terletak di kecamatan Kedungbanteng, kabupaten Banyumas di Jawa Tengah. Saya sangat menyukai Pemrograman dan Ilmu Komputer. Kadang saat sedang jenuh untuk memrogram, saya biasanya mendengarkan lagu atau membuat ilustrasi sederhana menggunakan media digital.

Saya mulai mengenal pemrograman di Sekolah Dasar akhir. Saat itu saya baru tahu sekilas tentang pemrograman. Lalu sekitar kelas 9 SMP, saya mulai belajar bahasa pemrograman pertama saya. Singkatnya saya terus belajar dan sebisa mungkin mempraktekkan apa yang saya pelajari sampai saat ini.
